# LAB-02: Operators and Expressions

**Duration:** 30 minutes  
**Points:** 4  
**Due:** Tuesday Week 1, End of Class

---

## 🎯 LEARNING OBJECTIVES

- Use arithmetic operators (+, -, *, /, %)
- Use comparison operators (===, !==, <, >)
- Use logical operators (&&, ||, !)
- Combine operators in expressions

---

## 📋 REQUIREMENTS

Create a JavaScript file that demonstrates:

1. **Arithmetic Operations:**
   - [ ] Calculate the sum of two numbers
   - [ ] Calculate the product of two numbers
   - [ ] Calculate the remainder of division (modulo)
   - [ ] Log results with descriptive labels

2. **Comparison Operations:**
   - [ ] Compare if two numbers are equal
   - [ ] Compare if one number is greater than another
   - [ ] Log the boolean results

3. **Logical Operations:**
   - [ ] Use AND (&&) operator
   - [ ] Use OR (||) operator
   - [ ] Log the boolean results

---

## 💻 EXAMPLE OUTPUT

```
Product: 50
Is Equal: false
Is Greater: true
Both True: true
```
