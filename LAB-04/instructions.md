# LAB-04: Introduction to Functions

**Duration:** 30 minutes  
**Points:** 4  
**Due:** Thursday Week 1, End of Class

---

## 🎯 LEARNING OBJECTIVES

- Declare and call functions
- Use function parameters
- Return values from functions
- Understand function scope

---

## 📋 REQUIREMENTS

Create a JavaScript file that demonstrates:

1. **Simple Function:**
   - [ ] Create a function that greets a user by name
   - [ ] Takes name as parameter
   - [ ] Logs greeting message

2. **Function with Return:**
   - [ ] Create a function that adds two numbers
   - [ ] Takes two parameters
   - [ ] Returns the sum

3. **Function with Logic:**
   - [ ] Create a function that checks if a number is even
   - [ ] Takes number as parameter
   - [ ] Returns true/false

4. **Call All Functions:**
   - [ ] Call each function with test values
   - [ ] Log the results

---

## 💻 EXAMPLE OUTPUT

```
Hello, Alice!
Sum: 15
Is 8 even? true
Is 7 even? false
```
