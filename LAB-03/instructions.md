# LAB-03: Control Flow - If/Else Statements

**Duration:** 30 minutes  
**Points:** 4  
**Due:** Wednesday Week 1, End of Class

---

## 🎯 LEARNING OBJECTIVES

- Write if/else statements
- Use else if for multiple conditions
- Make decisions based on conditions
- Control program flow

---

## 📋 REQUIREMENTS

Create a JavaScript file that demonstrates:

1. **Simple If/Else:**
   - [ ] Check if a number is positive or negative
   - [ ] Log appropriate message

2. **Multiple Conditions (else if):**
   - [ ] Check a student's score (0-100)
   - [ ] Assign letter grade (A, B, C, D, F)
   - [ ] Log the grade with the score

3. **Nested Conditions:**
   - [ ] Check if someone can vote (age >= 18)
   - [ ] If yes, check if they're registered
   - [ ] Log appropriate message

---

## 💻 EXAMPLE OUTPUT

```
The number 15 is positive
Score 85: Grade B
You are eligible to vote and registered!
```
